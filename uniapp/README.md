## 分享功能
* 小程序分享不能手动触发，只能用户点击分享按钮
* 实现分享：现在分享回调生命周期函数`onShareAppMessage`
* 返回分享配置：
    * path 路径
    * imageUrl 主图（不能太多并且白名单域名）
    * title 标题
    * desc 描述
