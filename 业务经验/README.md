## 多端设备自适应和响应式
* 自适应：
    * 多套代码，对应专门的设备
    * 根据多个域名，对应的页面
    * 功能多，布局复杂，交互多，适合自适应，通过浏览器查看设备是手机，平板，或者PC
* 响应式：
    * 一套代码，多端适配。
    * 定位一个域名，适应任何宽度
    * 功能减低，内容少，交互少，使用响应式，根据宽度的变化，页面自己适应。媒体布局
