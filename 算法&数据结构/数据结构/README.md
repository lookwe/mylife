## Object与Map
* 使用场景
    * Object：
        1. 仅做数据存储，并且属性仅为字符串或Symbol；
        2. 需要进行序列转换JSON传输；
        3. 当一个对象实例，需要保存属性，以及方法；
    * Map：
        1. 需要频繁更新，修改，删除，键值对；
        2. 存储大量数据，尤其key类型未知的情况下；
        3. 需要频繁进件迭代遍历处理；
