## 栈、队列、数组
* 数组-Array
    * 概率：存储某一串东西的集合，例如：一个弹夹，一袋米
    * static Array：  静态数组
        * 固定长度
        * 读写操作都是O(1)
    * dynamic Array： 动态数组
        * 不固定长度，每次扩容其实是，新建更大存储容器 + 拷贝原有内存元素进去
    * 时间复杂度：
        * 读取： O(1)
        * 写入：O(1)
        * 插入：O(n)  插入， 后面元素 往后落位，可能触发扩容
        * 删除：O(n)  删除后需要重新排列位置
        * 删末尾：O(1)
        * 新增：O(1) 可能触发扩容
        * 扩容：O(n)
