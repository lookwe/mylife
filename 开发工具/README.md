## VSCode开发工具篇
* 隐藏项目部分文件
    1. 创建 `.vscode/settings.json` 文件
    2. 配置选项 `files.exclude`
    ```java
        "files.exclude": {
            "**/.git": true,
            "**/.svn": true,
        }
    ```
