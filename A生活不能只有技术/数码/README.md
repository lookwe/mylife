        * 性价比、色彩好、部分领域擅长支持高
    * N卡 英伟达
        * AI算计强、市场心态广、游戏调节补帧光追强、生产力高
    * 国产
        * 发展阶段
* 购买入坑指南
    * 因高精度原件和高昂价格收获需谨慎，拆封很难退款
    * 拍视频一镜到底，包括快递包不剪辑不快进
    * 检查防擦贴、防静电贴、风扇灰尘、保护膜
    * 查SN码质保时间
    * 要支持个人送保
